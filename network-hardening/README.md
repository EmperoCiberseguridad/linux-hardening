# Network Hardening

Este proyecto documenta buenas prácticas para **hardening de red y firewall en Linux**, incluyendo `iptables` y `ufw`.  
Su objetivo es reducir la exposición a ataques y proporcionar ejemplos prácticos para entornos productivos.

## Objetivos

- Configuración segura de firewalls
- Ejemplos de reglas para servicios comunes
- Monitoreo de conexiones y puertos abiertos

## Alcance

- UFW
- iptables
- Monitoreo básico de red

## Estructura del proyecto

- `ufw-guide.md` → Guía de configuración segura de UFW  
- `iptables-guide.md` → Guía de configuración segura de iptables  
- `scripts/firewall-check.sh` → Script educativo para verificar reglas y conexiones
