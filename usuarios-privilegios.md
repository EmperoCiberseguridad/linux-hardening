# Usuarios y Privilegios

Este documento describe buenas prácticas para la gestión de usuarios y privilegios en sistemas Linux, con el objetivo de reducir riesgos asociados a accesos indebidos o abuso de permisos.

## Objetivo

Garantizar que los accesos al sistema estén claramente definidos, sean mínimos y auditables.

## Buenas prácticas

- Evitar el uso cotidiano del usuario root
- Asignar cuentas individuales a cada usuario
- Aplicar el principio de menor privilegio
- Revisar periódicamente usuarios activos
- Eliminar cuentas innecesarias o inactivas

## Privilegios administrativos

- Limitar el uso de sudo
- Registrar actividades administrativas
- Evitar privilegios permanentes innecesarios

## Alcance

Aplica a sistemas Linux multiusuario en entornos productivos o de laboratorio.
